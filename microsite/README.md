# Project Documentation for Try2

This project is a microsite for the 49. PS Kočovníci, designed to provide information about the organization, its activities, and its history. The site is structured to ensure easy navigation and a consistent user experience across different sections.

## Project Structure

The project contains the following files:

- **index.html**: The main entry point of the application. This file remains unchanged and serves as the homepage for the microsite.

- **troops.html**: This file contains information related to "Družiny" (Troops). It features the same navigation bar as the index.html file, ensuring users can easily navigate back to the homepage, camps, photos, what we do, and history.

- **camps.html**: This file provides details about "Tábory" (Camps). It maintains the same navbar structure as the index.html file for consistency across the site.

- **photos.html**: This file showcases "Fotky" (Photos) from various events and activities. It includes the same navigation bar as the index.html file for seamless navigation.

- **whatwedo.html**: This file outlines "Co umíme" (What We Do), detailing the activities and skills the organization focuses on. It features the same navbar structure as the index.html file.

- **history.html**: This file presents the "Historie" (History) of the organization, detailing its background and milestones. It retains the same navbar structure as the index.html file to ensure a cohesive user experience.

## Navigation

Each HTML file includes a navigation bar with links to the following sections:

1. Home Page (News, About Us, Event Plan)
2. Troops
3. Camps
4. Photos
5. What We Do (Sections)
6. History

This structure allows users to easily access different parts of the site without confusion, enhancing the overall usability of the microsite.
